# pinterest-ads-cli

A Pinterest Ads CLI designed for AI agents. Wraps the official Pinterest Ads API with simple, agent-friendly commands.

**Works with:** OpenClaw, Claude Code, Cursor, Codex, and any agent that can run shell commands.

> **Status: Under Development** — This project is not yet functional. Star/watch for updates.

## API Reference

- Official docs: https://developers.pinterest.com/docs/api/v5/
